# L'Épicure

A responsive luxury fine-dining website built with Next.js, TypeScript, and Tailwind CSS. It features a cinematic intro, tasting-menu experience, à la carte menu, restaurant story, gallery, and reservation request flow.

## Tech stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run build
```

## Notes

The reservation form currently provides a frontend demonstration flow and is not connected to a booking backend.
